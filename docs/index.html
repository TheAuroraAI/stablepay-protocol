What It Does

Multi-Sig Vaults

M-of-N approval threshold for every transfer. Institutional-grade custody without a custodian.

Compliance Controls

Pause/unpause, transfer limits, and allowlist enforcement — all on-chain, all auditable.

AI-Native via MCP

6 MCP tools let AI agents create vaults, propose transfers, and query state without writing Solana code.

x402 Payment Rails

Write operations are gated behind USDC micropayments on Base via x402 — agents pay per call.

MCP Tools (AI Agent Integration)

stablepay_initialize_vault

Create a new multi-sig vault with threshold, signers, and compliance settings.

stablepay_propose_transfer

Propose a token transfer from vault to recipient.

stablepay_approve_transfer

Approve a pending transfer proposal (signer only).

stablepay_execute_transfer

Execute a proposal that has reached approval threshold.

stablepay_get_vault

Fetch vault state: balance, signers, threshold, pause status.

stablepay_list_proposals

List all transfer proposals with status and approval counts.

Quick Start

npm install @stablepay/sdk

import { StablePaySDK, findVaultPda } from '@stablepay/sdk';

const sdk = new StablePaySDK(connection, wallet);

// Create 2-of-3 multisig vault
const vaultPda = await sdk.initializeVault({
  threshold: 2,
  signers: [signer1, signer2, signer3],
  transferLimit: new BN(100_000_000), // 100 USDC
});

// Propose transfer
await sdk.proposeTransfer(vaultPda, {
  recipient: recipientPubkey,
  amount: new BN(50_000_000),
  mint: USDC_MINT,
});
